Solution Method

This calculator uses Gaussian elimination with partial pivoting to solve the system of linear equations.

The method follows these steps:

  1. Set up the augmented matrix [A|b] where A is the coefficient matrix and b is the constant vector
  2. Use row operations to transform the matrix into row echelon form
  3. Use back-substitution to find the values of the variables
